# samSiahaan(); - Personal Portfolio Website 🚀

A modern, responsive, and interactive personal portfolio website built to showcase my professional journey, technical projects, and certifications. 

![Live Demo](https://img.shields.io/badge/Live%20Demo-Online-success?style=for-the-badge)
🌐 **Live Website:** [https://samsiahaan.vercel.app](https://samsiahaan.vercel.app)

---

## ✨ Features

* **📱 Fully Responsive Design:** Optimized for all devices (Mobile, Tablet, and Desktop) using Tailwind CSS.
* **💼 Dynamic Career Journey:** Interactive professional experience timeline with expand/collapse details (Responsibilities, What I Learned, and Impact).
* **📁 Project Showcase:** Comprehensive portfolio section with project thumbnails, tech stack details, GitHub/Demo links, and a dynamic "Read more" text-clamp feature.
* **📬 Integrated Contact Section:** Stunning gradient social media cards and a built-in direct email messaging form.
* **⚡ Smooth Navigation:** Fast and seamless page transitions powered by Vue Router.

---

## 🛠️ Tech Stack

* **Framework:** [Vue.js 3](https://vuejs.org/) (Vue CLI)
* **Routing:** [Vue Router](https://router.vuejs.org/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Icons:** Heroicons & Custom SVGs
* **Deployment:** [Vercel](https://vercel.com/)
